Boreal Caribou Monitoring Toolkit https://www.cclmportal.ca/resource/boreal-caribou-monitoring-toolkit

Currently, the Monitoring Working Group is creating a toolkit to help users select the boreal caribou monitoring methods which may be applied to their program and/or re-evaluate the methods currently being used in their area. The objectives of this toolkit are:

  • To centralize existing resources relevant to the monitoring of boreal caribou,

  • To provide valuable and accessible guidance for anyone engaged in, or planning to initiate, monitoring programs, and

  • To identify where knowledge gaps still exist, such that toolkit users may see opportunities to contribute resources to the toolkit.

Be sure to read more about the Monitoring Working Group’s ongoing toolkit project by clicking https://www.cclmportal.ca/project/monitoring-working-group-nbckc-project-page.
